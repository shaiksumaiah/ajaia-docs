## 📽️ The Walkthrough Script (Word-for-Word)

*(Open [https://ajaia-docs-mz2a-coral.vercel.app/](https://ajaia-docs-mz2a-coral.vercel.app/) before starting)*

### 1. Introduction
"Hi, I'm [Your Name], and I'm excited to present my submission for the Ajaia Collaborative Editor. I've built a full-stack document editor using Next.js 15, Tiptap, and Supabase."

### 2. The Dashboard & Document Creation
"On the Dashboard, we can see our existing documents. I'll click **'New Document'**. This is instantly saved to our **Supabase Cloud database**. I can rename it here to 'Ajaia Project Notes'."

### 3. Rich-Text Editing
"The editor supports full rich-text formatting. I can make text **Bold**, *Italic*, or __Underlined__. I can also add headings and lists. Everything auto-saves as I type."

### 4. File Import & Sharing
"I can also **Import** existing files. I'll select a `.txt` file, and noticed how it's instantly converted into a new document. 
Finally, for **Sharing**, I can click 'Share' and add `bob@example.com`. Bob now has access, and this is tracked in our cloud database."

### 5. Technical Decisions
"I chose a hybrid approach for persistence: **Supabase** for the live deployment to ensure data stays saved in the cloud, and a **local JSON database** for easy zero-config testing by any developer."

## 📦 Submission Step-by-Step (The "Spoon-feed")

1.  **Record your Screen:** Follow the script above using your screen recorder.
2.  **Zip the Folder:** Right-click your `AjaiaLLC-assesment` folder and choose **"Compress to ZIP"**.
3.  **Upload to Drive:** 
    - Go to Google Drive.
    - Create a folder: `Ajaia_Submission_[YourName]`.
    - Upload the ZIP file.
    - Create a small text file with your **Live URL**, **GitHub Link**, and **Video Link**.
4.  **Share:** Set the Drive folder to **"Anyone with the link"** and send it!
